# 知识库建设方案 — 数据来源、采集与整理

> 最后更新：2026-04-11
> 关联文档：[DESIGN.md](./DESIGN.md)（整体产品设计）

---

## 一、总体思路

知识库四层结构，每层的数据特征完全不同：

| 层级 | 数据性质 | 来源类型 | 能否自动化 |
|------|---------|---------|-----------|
| 第1层：地区政策 | 结构化事实 | 政府公开文件 | 高度自动化 |
| 第2层：学科考纲 | 半结构化 | 课标+考试说明+真题反推 | 中度自动化 |
| 第3层：诊断标准 | 专家经验 | 教师经验+数据统计 | 低，需人工 |
| 第4层：学习路径 | 专家经验+资源整合 | 教师设计+资源库 | 低，需人工 |

**核心原则：第1-2层靠爬取+LLM提取，第3-4层靠教师+LLM辅助生成，缺一不可。**

## 二、第1层：地区政策库

### 2.1 需要什么数据

每个地级市（中考以地级市为单位命题）的：
- 考试科目及各科满分
- 考试形式（闭卷/开卷/机考/实验操作）
- 考试时长
- 特殊政策（如体育分值、英语听力/口语占比、加分政策）
- 各区差异（如北京各区虽然统考，但教研重点和模拟题风格不同）

### 2.2 数据从哪来

**一手来源（权威，每年更新）：**

| 来源 | 获取方式 | 示例 |
|------|---------|------|
| 各地级市教育局官网 | 每年3-4月发布中考实施方案 | `site:gov.cn "XX市" "中考" "实施方案" 2026` |
| 省级教育考试院 | 发布全省统一政策框架 | 北京 https://www.bjeea.cn/ 、上海 https://www.shmeea.edu.cn/ |
| 教育部官网 | 国家层面指导意见 | http://www.moe.gov.cn/ |

**二手来源（辅助验证）：**
- 中考网 https://www.zhongkao.com/ — 按城市整理政策，更新快
- 各地教育类公众号

### 2.3 怎么采集

```
步骤1：确定目标城市列表
       MVP阶段 → 只做北京（4个主要城区：海淀/西城/东城/朝阳）
       
步骤2：人工找到该市当年中考方案的官方文件URL
       搜索 site:gov.cn "北京市" "初中学业水平考试" 2026
       
步骤3：用 LLM 从政策文件中提取结构化数据
       输入：政策文件全文（PDF/网页文本）
       提示词：从以下中考政策文件中提取结构化信息，输出YAML格式，
              包含：考试科目、各科满分、考试形式、时长、特殊规则...
       输出：结构化 YAML
       
步骤4：人工校验（关键！政策不能出错）
```

### 2.4 更新频率

- 每年一次（3-4月各地出新政策后更新）
- 大部分城市年际变化不大，主要关注改革年份

## 三、第2层：学科考纲库

### 3.1 需要什么数据

每个科目的：
- 知识点清单及层级结构（章 → 节 → 知识点）
- 各知识点在中考中的权重/分值占比
- 题型分布（选择/填空/解答/实验/作文...）
- 各题型的分值和数量
- 难度分布（基础:中档:难题 的典型比例）

### 3.2 数据从哪来

这一层需要**三个来源交叉验证**：

**来源A：国家课程标准（知识点骨架）**

```
文件：《义务教育XX课程标准（2022年版）》（共10个学科）
获取：教育部官网有PDF，也可从京东/当当购买纸质版（约25元/本）
价值：提供了"学什么"的完整清单和层级结构
局限：不告诉你"考什么、怎么考、各占多少分"
```

2022版新课标是命题的最终依据（教育部已明确"依标命题，不再制定考试大纲"），所以这是知识点体系的根基。

**来源B：各地考试说明/命题指导（考什么、怎么考）**

```
现状：2022新课标后，部分城市仍发布考试说明（换了名字），部分已取消
北京：仍每年发布《考试说明》，可从考试院购买
其他城市：需逐一确认
获取方式：考试院官网、学校教师渠道
```

**来源C：历年真题反推（最真实的权重数据）**

```
这是最可靠的权重来源！——考试说明是纲领，真题才是现实。

做法：收集目标城市近3-5年中考真题，逐题标注知识点，统计：
  - 每个知识点出现的频次
  - 每个知识点对应的分值总和
  - 题型分布实际数据
  
数据源：
  - 菁优网 https://www.jyeoo.com/ （结构化最好，每题已标注知识点，VIP约200元/年）
  - 学科网 https://www.zxxk.com/ （覆盖最广，年费约298元）
  - 学科网组卷系统 https://zujuan.zxxk.com/ （按知识点/年份/地区筛选）
```

### 3.3 怎么采集和整理

```
步骤1：获取知识点骨架
       来源：2022版课程标准
       方法：用 LLM 从课标PDF中提取知识点层级结构
       输出：每个学科一棵知识点树（YAML格式）
       
步骤2：获取考试权重
       来源：菁优网/学科网的真题知识点标注
       方法A（推荐）：利用菁优网已有的知识点分类浏览功能，
              直接提取其知识点树结构（网页端免费可见）
       方法B：收集近5年真题PDF，用 LLM 逐题标注知识点，统计权重
       输出：知识点树 + 每个节点的分值权重
       
步骤3：整理题型分布
       来源：真题卷面结构
       方法：分析近5年真题的固定题型结构（通常很稳定）
       输出：题号→题型→分值→对应知识模块 的映射表
       
步骤4：合并校验
       将课标知识点树 × 真题权重 × 题型分布 合并为最终的学科考纲
       人工校验，特别关注：
       - 课标有但近年不考的点（标记"了解即可"）
       - 近年高频但课标不强调的点（可能是地方特色）
```

### 3.4 关键数据平台对比

| 平台 | 优势 | 劣势 | 费用 | 建议用途 |
|------|------|------|------|---------|
| 菁优网 | 知识点标注最精细，结构化好 | 覆盖地区有限 | ~200元/年 | 知识点体系+权重分析 |
| 学科网 | 覆盖面最广，更新最快 | 知识点标注不如菁优细 | ~298元/年 | 真题/模拟题采集 |
| 国家智慧教育平台 | 免费，官方资源 | 无结构化题库 | 免费 | 教学视频资源引用 |

**MVP阶段建议：先买菁优网VIP，它的知识点树可以直接作为我们第2层的骨架。**

## 四、第3层：诊断标准库

### 4.1 需要什么数据

每个知识模块的：
- 水平分级定义（L0/L1/L2/L3 各是什么表现）
- 每个级别的判定信号（考生做什么样的题、得什么分，判定为哪个级别）
- 入门诊断测试题（快速定级用）

### 4.2 数据从哪来

**这一层没有现成数据源，必须构建。** 但可以从以下途径加速：

**途径A：教师经验提炼（核心）**
```
找 2-3 个有经验的中考数学/英语/物理老师
访谈问题：
  - "函数这个板块，你怎么判断一个学生是什么水平？"
  - "你见过的学生，在函数上通常卡在哪几个坎上？"
  - "从60分提到80分和从80分提到100分，学生面临的问题有什么不同？"
  
将访谈内容结构化为水平分级定义
```

**途径B：真题难度分层反推**
```
以数学函数模块为例：
  1. 收集近5年北京中考+海淀/西城一模中所有函数相关题
  2. 按难度排序（利用菁优网的难度标注，或根据得分率）
  3. 定义：能做对最简单的一组 = L1，能做对中档 = L2，能做对压轴 = L3
  4. 这样每个level就有了对应的"标杆题目"
```

**途径C：LLM 生成初稿 + 教师审核**
```
提示词：你是一位有10年经验的北京中考数学老师。
请为"二次函数"这个知识模块定义4个水平等级（L0-L3），
每个等级描述学生的典型表现，并给出2-3道可以用于判定该等级的诊断题。

LLM 输出的初稿质量通常在70-80%，教师审核修正后即可使用。
```

### 4.3 整理方式

```
步骤1：LLM 基于课标和真题，生成每个知识模块的水平分级初稿
步骤2：教师审核修正（重点：分级界限是否合理，判定信号是否准确）
步骤3：为每个级别匹配 3-5 道诊断题（从真题库中选取）
步骤4：小范围测试（找几个学生实测，看诊断结果是否符合老师判断）
```

## 五、第4层：学习路径库

### 5.1 需要什么数据

每个知识模块 × 每个水平等级，需要一条完整的学习路径：
- 学习步骤序列（先学什么后学什么）
- 每步使用的资源（具体到教材哪章、参考书哪个专题、哪个视频）
- 每步的学习方法（怎么用这个资源）
- 过关标准（做到什么程度算学会了）
- 常见错误和预警（易踩的坑）

### 5.2 数据从哪来

**来源A：教辅书的既有体系（资源骨架）**
```
主流教辅已经帮我们做了很多组织工作：
  - 五三（五年中考三年模拟）：按教材章节，知识清单→考点突破→真题训练
  - 万唯中考：按考点组织，真题分类汇编
  - 天利38套：纯真题训练
  
获取方式：
  - 京东/当当商品详情页有完整目录截图
  - 购买实体书翻拍目录（几十元成本）
  - 这些目录就是"推荐什么资源"的基础
```

**来源B：国家智慧教育平台（免费视频资源）**
```
https://www.smartedu.cn/
  - 覆盖所有学科的教学视频，按课标章节组织
  - 免费、官方、质量有保障
  - 可以作为每个学习步骤的视频资源推荐
```

**来源C：教师设计（学习方法和过关标准）**
```
这是最核心的部分，没有现成数据源。

需要教师回答的问题：
  - "一个函数L0的学生，你会让他先做什么？"
  - "你推荐用五三还是万唯？五三的哪些题适合这个水平？"
  - "怎么判断他函数学过关了？"
  - "这个阶段学生最容易犯的错误是什么？"
  
一个老师覆盖1个学科，3个老师覆盖数学+英语+物理（MVP的3科）
```

**来源D：LLM 扩展和个性化**
```
教师提供的是"骨架路径"（每个模块一条标准路径）
LLM 负责：
  - 根据考生具体情况微调（如已经会了某步骤则跳过）
  - 生成自然语言的学习指导文本
  - 根据考生反馈动态调整建议
```

### 5.3 资源推荐的约束规则

```yaml
resource_rules:
  safe_to_recommend:  # 可以放心推荐
    - "对应版本教材（人教版/北师大版等，需匹配考生所用教材）"
    - "五年中考三年模拟（五三）"
    - "万唯中考系列"
    - "天利38套"
    - "国家中小学智慧教育平台视频"
    - "各区一模二模真题"
    - "近5年本市中考真题"
  
  need_verification:  # 需要确认后才推荐
    - "各类网课平台（质量参差）"
    - "小众参考书"
    - "自媒体教学内容"
    - "AI生成的练习题"
  
  never_recommend:  # 不推荐
    - "盗版资源"
    - "未经验证的付费课程"

  matching_rules:  # 资源匹配规则
    - "L0/L1 学生 → 只推荐基础题，跳过★★★难题"
    - "L2 学生 → 中档题为主，适量接触难题"
    - "L3 学生 → 真题+模拟题限时训练为主"
    - "教材版本必须匹配考生所在地区使用的版本"
```

### 5.4 整理方式

```
步骤1：建立资源库
       整理可推荐的资源清单（教材+教辅+视频+真题）
       每个资源标注：适用学科、适用章节、适用水平、获取方式
       
步骤2：教师设计标准路径
       每个知识模块 × 每个起始水平 = 一条路径
       教师提供：步骤序列、推荐资源、过关标准、常见错误
       
步骤3：LLM 扩展路径细节
       基于教师骨架，生成详细的学习指导文本
       教师审核修正
       
步骤4：建立"过关测试"题库
       从真题库中为每个过关标准选取 5-10 道匹配题目
       用于判断考生是否通过该步骤
```

## 八、MVP 阶段具体执行计划

### 范围：北京市 × 数学（1个城市 × 1个学科，先跑通全链路）

| 步骤 | 任务 | 数据源 | 方法 | 预计耗时 |
|------|------|--------|------|---------|
| 1 | 北京中考政策整理 | 北京教育考试院 | 人工查找+LLM提取 | 0.5天 |
| 2 | 数学课标知识点树 | 2022版数学课标PDF | LLM提取+人工校对 | 1天 |
| 3 | 北京数学考试权重 | 菁优网+近5年真题 | LLM标注+统计 | 2天 |
| 4 | 数学各模块诊断标准 | LLM初稿+教师审核 | 先生成后审核 | 2天（含等待教师反馈） |
