## 本地预览

### 1. 安装微信开发者工具

下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目

1. 打开微信开发者工具
2. 点「小程序」→「+」→「导入项目」
3. **项目目录**：选 `/Users/jiakui/projects/zhongkao-agent/miniprogram/`
4. **AppID**：暂选「测试号」（不需要真实 AppID）
5. **后端服务**：暂不使用云开发（本原型纯前端）
6. 点击「确定」

### 3. 预览效果

- 默认进入群聊页面
- 可滚动查看完整 mock 对话（班主任建档对话 → 任务推送 → 学生发题 → 数学老师讲题）
- 可在输入框打字、点发送，会触发"班主任 mock 回复"
- 可点击解题卡片，弹出"正式版跳转详细解析页"提示

## 技术选型

- **框架**：微信小程序原生开发（避免 Taro 等中间层在群聊高频更新场景的坑）
- **UI 组件**：自建群聊组件（不用现成 IM SDK，因为消息发送方是 AI Agent）
- **后端通信**（待接入）：HTTP/JSON + 长连接
- **状态管理**：原生 setData（数据量小，不需要 MobX 之类）

完成上述步骤后，将本目录的代码：
1. 在 `project.config.json` 替换 `appid` 为正式 AppID
2. 在 `app.js` 加上 `wx.cloud.init({ env: 'xxx' })`
3. 把 `pages/chat/chat.js` 里的 `MOCK_MESSAGES` 替换为真实 API 拉取
4. 接入 `../backend/` 提供的对话/推送/订阅消息接口

## 与 backend 目录的关系

- 本目录：小程序前端（微信原生代码）
- `../backend/`：Node.js 后端（多 Agent + API + 推送调度）
