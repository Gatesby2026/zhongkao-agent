## 这是什么

家长打开微信小程序后看到的「私人教研组」群聊 UI 主对话载体。

详见根目录 `PRD.md` v5.0「四、产品形态」章节。

## 启动前置条件

按顺序：

1. ✅ 已有营业执照
2. ⏳ 注册微信服务号「中考私人教研组」+ 企业认证
3. ⏳ 注册同名微信小程序（同一企业主体）
4. ⏳ 申请「2026 AI 小程序成长计划」拿到 1 亿混元 token + 6 个月免费云开发
5. ⏳ 服务号 ↔ 小程序 unionid 打通

完成上述步骤后才开始本目录的代码开发。

## 技术选型

- **框架**：微信小程序原生开发（避免 Taro 等中间层在群聊高频更新场景的坑）
- **UI 组件**：自建群聊组件（不用现成 IM SDK，因为消息发送方是 AI Agent）
- **后端通信**：HTTP/JSON + 长连接（接订阅消息走云开发触发器）
- **状态管理**：MobX-miniprogram 或原生 setData

## 与 backend 目录的关系

- 本目录：小程序前端（微信原生代码）
- `../backend/`：Node.js 后端（多 Agent + API + 推送调度）
