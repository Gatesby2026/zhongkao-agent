# 2026 录取位次预估 · 整体设计方案

> 目标:给朝阳**全部 89 所**高中(公办/民办/中职/贯通/统筹/新校)各一个 **2026 预估录取位次**,
> 作为志愿填报核心依据。每个预估 = 点估 + 区间 + 百分位 + 方法标签 + 可信度,**官方为锚、估算标注、绝不伪精确**。
> 本文是设计,不含模型代码。数据来源见 `knowledge-base/admission/beijing/evidence/`。

---

## 一、为什么 2026 必须重新预估(不是照搬历史线)
三个叠加变量,使 2026 与往年**不可直接比**:
1. **生源高峰**:朝阳参加中考 2025=10729 → 2026≈12000(+~12%);全市初三 130701(+13.2%)。
2. **普高大扩招**:全市 8.5万→9.5万(+1万),普职比 2.58→**2.88**;朝阳公办统招盘子↑ + 新增 4 校。
3. **提前批扩招**:全市 1+3 8795→>1万、0.5+3 直升破万;但**贯通缩**(1035→910)。
→ 同一所校,池子大小和竞争人数都变 → **录取位次会系统性移动**,必须建模而非套旧线。

---

## 二、核心洞察:决定录取位次的是「**剩余统招盘子**」,不是总招生
本次逐校采集已证实(见 `chaoyang-school-plan-table`):
```
某校统招录取位次(cutoff) ≈ 在"统招竞争池"里,排到该校【剩余统招】席位填满处
剩余统招 = 总招生 − 校额到校 − 提前批(直升/1+3/贯通/特长)
```
**实例(2025)**:陈经纶总610,但校额95+提前批280 → **剩余统招仅235**;朝外总387,提前批305 → **剩余统招仅82**。
→ **集团龙头校(朝外/人朝/清华朝阳/陈经纶/八十)近半~八成被提前锁走,统招门槛被显著抬高**。
→ 模型必须用**剩余统招**做分母,而非总招生。这是本设计区别于"套历史线"的关键。

## 四、模型:位次 → 百分位 → 2026 位次(分母可比)
**Step1 历年位次归一为百分位**(解决跨年基数不同):
```
百分位(年) = 录取位次(年) ÷ 当年参加中考人数(年)
  例:陈经纶 2025 位次2376 ÷ 10729 = 22.1%
```
**Step2 百分位趋势**(近年加权,看该校近年变化 = 用户第2点):
```
百分位_2026_基线 = 加权(p2023, p2024, p2025),近年权重高;波动大→区间放宽
```
**Step3 还原 2026 位次**:
```
位次_2026 = 百分位_2026 × 2026参加中考人数(≈12000)
```
**Step4 三档调整**(在基线上叠加):
- **A 普高扩招**:全区统招盘子↑→整体位次后移(门槛松);按全市普高 +11.8% 折算。
- **B 剩余统招变化**(关键):若该校 2026 提前批/校额↑→剩余统招↓→**该校位次前移(更难)**;反之后移。用逐校剩余统招(三、资产)算。
- **C 提前批掐尖**:1+3/直升扩招→统招池高分段被抽走→统招位次整体后移。用全市/朝阳提前批序列折算。
> A、C 是全局项(所有校同向),B 是逐校项(各校不同)。净效应 = 基线 × (1+A+C) 再按 B 微调。

## 五、分层预估(89 校,按数据可得性 4 档,统一输出)
- **Tier A 成熟公办(3年位次,主力~26校)**:四步法全用,可信度 T2-T3。
- **Tier B 新校(4校)**:无历史线 → 锚定法(同体系分校/本部 + 首年折让),沿用已建 est_rank(民大附2800/首师大实验3400/北大附3000/燕京新源6000),T3-T5。
- **Tier C 民办(19校)**:多数"门槛低基本可入"→位次靠后、区间宽;有网传线的用线,T3-T4。
- **Tier D 中职/贯通/统筹**:中职综高按综高线;贯通按≥380;统筹按统筹线(已有专业码/名额)。T3-T4。

## 六、提前批/扩招影响的量化(用户第4点,本设计已具备数据)
**机制**:统招池 = 全体考生 − 提前批锁定 − 校额锁定。
- 2026 提前批扩(1+3>1万/直升破万)→ 更多高分段在统招前被锁 → **统招 cutoff 位次后移**(数字变大、看似门槛降,实为池子变了)。
- **逐校差异**:朝外/陈经纶这类直升占比高的校,2026 若直升再扩,剩余统招更小 → 统招更难(位次前移);而直升少的校(日坛/十七/化工)受影响小。
- **量化路径**:① 全市提前批总量 Δ → 统招池高分段缩减比例 → 全局位次后移系数 C;② 逐校剩余统招 Δ → 逐校 B 调整。**数据已在库,可直接算**(贯通缩 910 也计入,方向相反)。

## 七、其他因素(第5点)
1. **大小年震荡**:去年线高→今年回避→线降,用波动项(line_trend.volatile)体现,勿被单年带偏。
2. **新校分流**:2026 新增 4 公办 → 分走周边生源 → 邻校位次略后移。
3. **改革断点**:2024(670)→2025(510)总分改 → **禁止跨年比分数**,一律百分位/位次。
4. **口径一致性**:剩余统招(③)、参加中考(分母)两个口径全程统一(本周已踩坑:总招生≠剩余统招、合计≠投朝阳)。
5. **可信度传播**:任何预估带 conf;低可信(新校/提前批估)显式标"区间+待核",UI 永不伪精确。
6. **出分前/后**:出分前用历史一分一段近似分数;出分后接当年一分一段→预估收敛为精确。

## 十、落地分期
- **P0**:建 `pred_2026` 字段 + Tier A 四步法(主力公办)+ 面板区块。用已采数据,先覆盖+可信度。
- **P1**:提前批/剩余统招逐校 B/C 调整定量;新校/民办/中职接入。
