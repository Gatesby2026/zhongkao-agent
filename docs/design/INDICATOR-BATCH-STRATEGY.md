## 一、官方规则(2026,硬事实)

1. 中招三批次:**① 提前招生 → ② 指标分配 → ③ 统一招生**,按批次顺序录取,**被前一批录取即锁定,后批作废**。
2. **校额到校与市级统筹同属"指标分配批",合用同一张志愿表,共 8 个志愿**(每志愿 1 校 + 最多 2 专业)。没有"校额几格、统筹几格"的分开配额,可任意混排、可不填满、可全空。
3. 录取机制:**分数优先、遵循志愿**——全市考生按中考总分从高到低排队;轮到某考生时,按其填报顺序 1→8 依次检索,名额已满则跳过看下一志愿,**跳过不扣分、不降档、不影响后续志愿**;8 个全未中则原样进入统招批。
4. 竞争口径不同:**校额到校**名额定向到初中,只与"本初中 + 也填了这所高中"的考生比(实质=中考总分在本校报名者内的排序,俗称校内排名);**市级统筹**按全市/区口径分数竞争。
5. 门槛:京籍应届 + 总分 ≥430/510 + 综合素质 B 等 + 同一初中连续三年学籍;非京籍/往届/回户籍/外省回京**不可报**整个指标批。
6. 数据事实:朝阳统筹清单只含**外区校**(本区优质校的统筹名额流向外区;朝阳孩子够本区校走统招/校额)→ 校额、统筹两列表**天然无重复校**,合并无需去重(此为数据模型不变量,若未来引入本区统筹校需加去重)。

## 二、核心决策模型

### 风险不对称(一切策略的根)

| 结果 | 后果 |
|---|---|
| 填了没中 | **零损失**,自动落统招 |
| 填了中了 | **锁定**,统招作废 |

### 唯一判准

> **"如果被这所学校锁定,是否比我统招的预期结果更满意?"** 是 → 填;否 → 空着。

由此推出四条铁律:

1. **宁缺毋滥**:绝不填 waste(统招本可达)与一般 similar(平级)——中了 = 主动锁进同级或更低校,是本批次最大的坑。**空格无任何惩罚**(区别于统招批的防死档填满)。
2. **应填尽填**:每一所真 worth = 一张免费彩票(不中无损失),有几所填几所,不要保守。
3. **"够不着"的校额也值得填**:校额竞争在校内、且只在同校报名者之间——名义上"够不着"的名额若本初中无人抢会直接掉下来(校额爆冷原理)。**统招位次视角的"够不着"对校额不成立**。统筹则相反:知分填报、全市按分竞争,够不着=死票,不填。
4. **真愿意去是前置关**:统筹多为外区远校+寄宿;档次再高、锁定了不愿去就是灾难。远郊寄宿校不进自动填,手动选需确认。

### 例外:同校保险

统招最想去的校 X 分数在边缘(可能差一点),而校额/统筹恰有 X 的名额 → 填 X 是**同一目标的更稳路径**,即使判为 similar 也值得填(锁进 X = 本来就想去 X,无损失)。

## 三、候选分类口径(实现规范)

输入:孩子区排 `R`(知分后为实际区排);高中 `ref`(统招录取位次,2025 实际线,数值越小档次越高)。

| 分类 | 判定 | 校额处置 | 统筹处置 |
|---|---|---|---|
| **worth** | `ref ≤ R×0.95`(统招够不上的 upgrade,**无上限**——顶级校也算) | ✅ 自动填 | 需再过统筹门槛(entry):够得着→✅ 自动填;够不着→❌ 死票不填 |
| **similar** | `R×0.95 < ref < R×1.1` | ❌ 不自动填(同校保险例外,待实现) | ❌ 不填 |
| **waste** | `ref ≥ R×1.1`(统招本可达) | ❌ 严禁 | ❌ 严禁 |
| 档次低于门槛 | 统筹校线 < 控制线 | — | ❌ 不值(走统筹≈460 反不如统招) |

通勤/住宿过滤(校额统筹通用):超通勤上限且不可住宿(或不接受住宿)→ 不自动填;统筹远郊(`j.far`)默认不自动填。

**口径假设(必须向用户展示的免责)**:校额的 worth/waste 用区排做 proxy,真实录取比**校内排名**,本工具不掌握校内排名 → 每张校额志愿都是"中签率未知的正 EV 彩票",不是承诺。

## 四、选择与排序算法

```
候选池 = 校额(本初中有名额的高中) ∪ 统筹(面向本区清单)
1. 门槛过滤:身份=京籍应届 且 ≥430 且 综评B;不满足 → 整批不可填,直接说明
2. 通勤/住宿过滤 + 统筹远郊剔除 + 统筹死票(够不上entry)剔除
3. 选择:所有 worth 校额 + 所有 worth且够得着 统筹    [现状]
   (+ 同校保险 similar;后续见 backlog)
4. 排序:按 ref 升序(档次好=更想去 → 在前);同档(same ref 档)校额优先于统筹
   理由:分数优先遵循志愿 → 梦想在前不损害现实志愿;校额优先因其
   校内竞争面小、本区近校、无住宿成本,同档下必优于统筹
5. cap = 8;超出取前 8 并告警"已按同档校额优先取前8"
6. 展示:每格 = 渠道标签(校额到校/统筹·统筹N) + 档位徽标 + 录取即锁定提示
   + 校额附"按校内排名录取,本工具不算校内排名"免责
```

### 分位次画像(自动填的预期形态)

- **高位次**(统招可达区内顶尖):worth 集近空 → 指标批可能全空,合理,提示"重心统招"。
- **中位次**:少量校额 worth + 个别统筹,形如 3-5 格。
- **低位次**:校额是最大逆袭通道,worth 名单长 → 尽量填满 8;统筹多为死票自动排除。
